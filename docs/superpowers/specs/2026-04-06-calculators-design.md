# Calculators Design Spec

**Date:** 2026-04-06

## Goal

Add two standalone calculator pages (`compound.html`, `goal.html`) and update the hub (`index.html`) to make them Live cards in the correct order.

## Common Conventions

- Language: Korean only, no i18n toggle
- Theme: Light mode only, no dark mode toggle
- Font: Inter (Google Fonts CDN, same as hub)
- Back link: `← 포트폴리오 홈` → `/` at top-left of each page
- Chart library: Chart.js (CDN: `https://cdn.jsdelivr.net/npm/chart.js`)
- All calculations run in-browser on slider `input` events (no server calls)
- Korean currency format: `₩X,XXX만` / `₩X.XX억` (formatted via JS)

### Sliders

| Label | Min | Max | Default | Step | Unit |
|---|---|---|---|---|---|
| 초기 투자금 | 0 | 100,000,000 | 10,000,000 | 1,000,000 | 원 |
| 연간 투자금 | 0 | 50,000,000 | 3,000,000 | 500,000 | 원 |
| 연 수익률 | 1 | 30 | 7 | 0.5 | % |
| 투자 기간 | 1 | 40 | 20 | 1 | 년 |

### Calculations

```
principal[0] = 초기 투자금
compound[y] = compound[y-1] * (1 + rate) + 연간 투자금   (y = 1..기간)
simple[y]   = 초기 투자금 + 연간 투자금 * y + 초기 투자금 * rate * y
straight[y] = 초기 투자금 + 연간 투자금 * y              (원금 선)

최종 금액    = compound[기간]
총 투자 원금 = 초기 투자금 + 연간 투자금 * 기간
순수 이자    = 최종 금액 - 총 투자 원금
원금 대비 배수 = 최종 금액 / 총 투자 원금
수익 비율    = 순수 이자 / 최종 금액 * 100
```

### Chart

- X축: 0년 ~ 투자 기간(년)
- Y축: 금액(원), 한국 포맷 레이블
- 데이터셋:
  - `복리 성장`: `#3b82f6` (파랑), fill 없음
  - `단리 성장`: `#6ee7b7` (연두), fill 없음
  - `원금`: `#9ca3af` (회색), borderDash `[6,4]`
- 툴팁: 마우스 hover 시 3개 라인 값 표시

### Sliders

| Label | Min | Max | Default | Step | Unit |
|---|---|---|---|---|---|
| 원금 (사전) | 0 | 500,000,000 | 60,000,000 | 1,000,000 | 원 |
| 연간 저축 | 0 | 50,000,000 | 10,000,000 | 500,000 | 원 |
| 투자 수익률 | 1 | 30 | 5 | 0.5 | % |
| 목표 금액 | 10,000,000 | 1,000,000,000 | 100,000,000 | 5,000,000 | 원 |

### Calculations

```
asset[0] = 원금(사전)
asset[y] = asset[y-1] * (1 + rate) + 연간 저축   (y = 1, 2, ...)
달성 연도 = 첫 번째 y where asset[y] >= 목표 금액

달성 시간(월 단위 보간):
  months = 달성 연도 * 12
  → 년 = Math.floor(months / 12), 개월 = months % 12

현재 자산   = asset[0] (= 원금)
월 필요액   = 연간 저축 / 12
자본률      = 원금(사전) / 목표 금액 * 100
총납입 금액 = 원금(사전) + 연간 저축 * 달성 연도
투자 수익   = asset[달성 연도] - 총납입 금액
수익 기여율 = 투자 수익 / asset[달성 연도] * 100

빠른 달성 가능 뱃지: 달성 연도 <= 10년이면 표시
목표 달성 불가: 수익률 0% 또는 연간 저축 0이고 원금 < 목표이면 "달성 불가" 표시
최대 계산 연도: 50년 (초과 시 "50년 이상" 표시)
```

### Chart

- X축: 0년 ~ min(달성 연도 + 3, 50)년
- Y축: 금액(원)
- 데이터셋:
  - `복리 자산`: `#3b82f6` (파랑)
  - `납입 원금`: `#6ee7b7` (연두)
  - `목표금액`: `#ef4444` (빨강), borderDash `[6,4]`, 수평선(목표 금액 상수)
- 달성 시점에 수직 점선 또는 annotation으로 표시 (Chart.js annotation 플러그인 대신 직접 수직선 data로 구현)

### Table

- 달성 연도까지 표시 (최대 50행)
- 달성 연도 행: 배경색 강조 (`#eff6ff`)
- 진행률 = asset[y] / 목표 금액 * 100 (100% cap)
- 진행률 바: CSS 인라인 width
